## What's included (ready-to-use)
- `index.html` — A static HTML file that mirrors the required **Home** page content:
  - H1 heading with project name
  - Paragraph with your name
  - Paragraph with date
  - Image block with alt text
  - List of WordPress benefits
  - Button linking to WordPress.org
- `media/homepage_screenshot.png` — placeholder image (replace with your actual homepage screenshot)
- `CHECKLIST.md` — step-by-step checklist for completing the assignment and documenting evidence (screenshots, notes).
- `INSTRUCTIONS.md` — commands and next steps to create the GitHub repo, clone it into AMPPS `www`, install WordPress, and push changes.
- `.gitignore` — minimal git ignore
- `post-getting-started.md` — sample post content you can paste into WordPress Posts.

---

## How you will use this
1. Create a GitHub repo named `ch08project_jhill172` (public).
2. Copy these files into your AMPPS `www` directory: e.g., `C:\Ampps\www\ch08project_jhill172`.
3. Use Softaculous in AMPPS to install WordPress into the `ch08project_jhill172` directory.
4. Follow the checklist to create the Home page inside WordPress, take the real screenshot, and replace `media/homepage_screenshot.png`.
5. Commit & push everything to GitHub.
